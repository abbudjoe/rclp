# Commercial Boundary

This repository is for the open protocol MVP only.

## Belongs here

- Protocol spec
- Reference implementation
- Demo agents
- Edge verifier
- Command-gate scaffold
- Conformance tests
- Isaac Sim POC scripts/docs
- Threat model
- Audit event schema

## Does not belong here

- Hosted trust root
- Customer accounts
- Hosted multi-tenant control plane
- Billing
- Customer account management
- Managed policy UI
- Fleet-scale audit backend
- Carrier/MVNO integrations
- Managed connectivity
- Production key management service
- Enterprise SSO
- Enterprise IAM
- Commercial SLAs
- Managed audit retention service
- Sales CRM
- Pricing logic
- Proprietary customer workflows

## Future commercial platform repository

The future commercial platform can implement:

- managed trust root
- customer accounts and billing
- policy lifecycle management
- managed policy UI
- fleet-scale audit/replay
- carrier/network adapters
- managed connectivity
- customer dashboards
- compliance retention
- enterprise SSO/IAM and integrations
- commercial SLAs
- hosted lease issuance and revocation

The open protocol should remain usable without the commercial platform.
